# Link in Bio - Codédex Project

This is a simple "Link in Bio" page built as part of the [Codédex](https://www.codedex.io/) final project challenge. It uses clean HTML and CSS to create a professional-looking landing page for social media links.

## Features
- **Profile Section**: Includes a profile picture, name, and a short "About Me" bio.
- **Link Buttons**: Styled buttons for various social media platforms and projects.
- **Responsive Design**: Centered layout that works well on both desktop and mobile screens.

## Technologies Used
- **HTML5**: For the structure of the page.
- **CSS3**: For styling, layout, and button designs.

## How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/Sailokesh-ctrl/link-in-bio-codedex.git
   ```
2. Open `index.html` in your web browser to view the page.

## Project Origin
This project was created following the requirements of the Codédex HTML/CSS course final project.
